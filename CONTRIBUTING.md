# Contributing to Patent Analyzer Prompts

Thank you for your interest in improving Patent Analyzer Prompts.

## How This Project Works

Patent Analyzer Prompts is a set of 6 markdown files containing carefully structured prompts for AI-assisted patent landscape research. There is no software to build or install — the prompts are copied directly into AI chat interfaces.

## How to Contribute

### Reporting Issues
- Use [GitHub Issues](https://github.com/scottconverse/patent-analyzer-prompts/issues) to report bugs, inaccuracies, or suggestions
- Include which stage prompt is affected and what the issue is
- If a prompt produces incorrect or misleading output, include the input you used and the problematic output

### Suggesting Changes
1. Fork the repository
2. Create a branch for your changes
3. Make your edits to the relevant stage markdown files
4. Test your changes by running the modified prompt in Claude or Gemini with a sample invention
5. Submit a pull request with a clear description of what changed and why

### Important Guidelines

**Legal Disclaimers:** All prompts must retain their embedded legal disclaimers and role disavowals. The AI must always be positioned as a research assistant, never as a lawyer or legal advisor. This is a license requirement (CC BY-SA 4.0 with UPL retention clause) and a legal necessity.

**Accuracy Rules:** Do not weaken the accuracy requirements (no fabrication, verified URLs, etc.). These protect users from AI hallucination.

**Plain Language:** All output instructions must maintain the plain-English writing style. The target audience is inventors and engineers, not lawyers.

**Testing:** Test any prompt changes with at least one sample invention description before submitting.
